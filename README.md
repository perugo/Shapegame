# 図形数当てゲーム

５種類の図形の数を当てるゲームです。

１０問出題されます。

![4](https://user-images.githubusercontent.com/34999008/214465737-04bf9aa0-9689-4868-9a5c-3a304eed69bc.png)



パネルは３×５マスで分かれており、１マスごとに何も置かないまたは●、■、▲、＋、★の図形をランダムで表示されます。


# 問題の種類

- 「●●はいくつですか？」 

- 「●●と△△を足して、いくつですか？」 

- 「●●から△△を引いて、いくつですか？」 

の三種類ある。どれかの問題が出題されます。プレイ回数ごとに、出題される問題が変化します。
